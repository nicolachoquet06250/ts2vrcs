Pour installer la bibliothèque, utilisez votre gestionnaire de paquets préféré :

```bash
echo "@nicolachoquet06250:registry=https://npm.pkg.github.com" >> .npmrc
```

```bash
npm install @nicolachoquet06250/ts2vrcs
```

Ou avec yarn :

```bash
yarn add @nicolachoquet06250/ts2vrcs
```

Ou pnpm :

```bash
pnpm add @nicolachoquet06250/ts2vrcs
```
