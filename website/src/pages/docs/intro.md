`@nicolachoquet06250/ts2vrcs` est une bibliothèque TypeScript pour transformer des scripts TypeScript en C#/U# compatible avec VRChat.

### Caractéristiques
- Support des types TypeScript complexes.
- Facile à intégrer.
- Hautement configurable.
